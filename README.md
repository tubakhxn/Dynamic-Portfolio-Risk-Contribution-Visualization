# Dynamic Portfolio Risk Contribution Visualization

## What is this project?
This project simulates a multi-asset portfolio and visualizes the dynamic risk contribution of each asset over time using a stunning, animated stacked area graph. It is designed for quantitative finance, risk management, and data visualization enthusiasts who want to:
- Understand how each asset contributes to total portfolio risk
- Explore the effects of time-varying volatility and dynamic portfolio weights
- Experience professional, hedge-fund-style risk visualization

The animation features:
- Correlated Geometric Brownian Motion simulation
- Rolling covariance and risk decomposition
- Smooth, glowing, dark-themed stacked area animation
- Advanced visual effects and real-time updates

3. **Install dependencies** (Python 3.10+ required):
   ```sh
   pip install -r requirements.txt
   ```

   ```sh
   python main.py
   ```
5. Enjoy the live animated risk contribution visualization!
